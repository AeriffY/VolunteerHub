为了实现异步发邮件, 请按以下操作配置:

首先修改 `.env` 文件:

```bash
MAIL_MAILER=smtp
MAIL_HOST= //填写你的smtp服务器地址, 如163邮箱是smtp.163.com, gmail是smtp.gmail.com
MAIL_PORT=	//填写端口. 163邮箱或qq邮箱是465, gmail是587
MAIL_USERNAME=	//填写你的邮箱地址, 不用加引号
MAIL_PASSWORD=	//在邮箱设置里开启POP3/SMTP等服务, 将生成一串授权码. 不用加引号
MAIL_ENCRYPTION=ssl
MAIL_FROM_ADDRESS=""	//你的邮箱地址, 要加引号
MAIL_FROM_NAME="志愿汇"
```

实在改不对的告诉我, 可以用我的.

修改该字段默认的sync为database:

```bash
QUEUE_CONNECTION=database
```



修改完成后, 在终端运行 `php artisan config:clear`

随后运行:

注意运行完最后一条命令后, 不要关闭终端

```bash
php artisan queue:table
php artisan migrate
php artisan queue:work
```
